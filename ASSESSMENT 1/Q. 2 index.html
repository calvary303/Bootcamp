TEXT FORMATTING TAGS OF HTML

Tags Description
<b> tag defines bold text
<i> tag defines text in alternate voice or mood
<u> tag underlines text written between the tag
<em> tag defines emphasized text
<big> tag defines bigger text
<small> tag defines smaller text
<strong> tag defines important text
<sub> tag defines subscripted text
<sup> tag defines superscripted text
<ins> tag defines inserted text
<del> tag defines deleted text
<mark> tag defines marked or highlighted text
